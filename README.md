# Monte Carlo Simulation for Investment Risk Analysis

## Overview

This project develops a Monte Carlo simulation framework to quantify uncertainty in investment decisions under stochastic market conditions.

## Features

- Monte Carlo Simulation
- NPV simulation
- IRR simulation
- Probability distribution analysis
- Tornado sensitivity analysis

## Technologies

Python • NumPy • Pandas • SciPy • Matplotlib
